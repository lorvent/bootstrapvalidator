BootstrapValidator v0.4.2 Released

New features

  • #168: Add siren and siret validators, thanks to @jswale
  • #177: Add vin validator
  • #184: Add excluded option

Improvements

  • #171: The phone validator now supports +1 country code and area code for US phone number, thanks to @tomByrer
  • #173: The remote validator allows to override name option, thanks to @jswale
  • #182: Improve zipCode validator, thanks to @gercheq

Bug fixes

  • #178: Do not validate fields that enabled is set to false, thanks to @henningda

Document

  • #169: Better to say: {validatorname} and {validatoroption} must be lowercase, thanks to @tomByrer

Contributors

Thank to the contributors:

  • @gercheq
  • @henningda
  • @jswale
  • @tomByrer
